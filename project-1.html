Gestão de Frotas

Projeto de gestão de frotas para uma empresa do setor de painéis solares. Desenvolvido na faculdade como estudo de caso, o projeto surgiu da necessidade real de uma empresa. A empresa precisava de informações para decidir entre alugar ou adquirir novos veículos para sua frota.

Resumo do desenvolvimento

O projeto surgiu da necessidade de uma empresa em decidir entre alugar ou adquirir novos veículos para sua frota. No entanto, a falta de dados fazia com que as decisões fossem baseadas em suposições. Identifiquei essa deficiência na gestão da informação e, devido ao orçamento limitado, utilizei formulários do Google para coletar dados no escritório e em campo. Com essas informações, desenvolvi análises no Power BI, demonstrando como o uso de dados pode apoiar decisões estratégicas.

O projeto utilizou dados ficticios, e foi apresentado para a empresa, que se mostrou interessada em implementar o sistema de coleta de dados e análises.

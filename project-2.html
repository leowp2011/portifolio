RPA com Python Selenium

O projeto foi solicitado pelo requerente para automatizar o processo de extração de dados de frete para tênis em diferentes sites de e-commerce, com base em informações fornecidas em uma planilha Excel. O objetivo é otimizar a coleta de dados, realizando a busca por modelos específicos de tênis e calculando os custos de frete para uma lista de CEPs previamente cadastrados. O requerente deseja obter os resultados de forma estruturada e eficiente, com os dados sendo armazenados em um arquivo Excel para facilitar a análise comparativa entre diferentes opções de frete e modelos de tênis.

Resumo do desenvolvimento

A partir da necessidade do requerente, foi desenvolvido um robô de automação utilizando Python e Selenium para realizar a coleta dos dados de frete de forma automatizada.

O script recebe como entrada uma planilha Excel com os modelos de tênis e os CEPs para os quais se deseja obter o valor do frete. Em seguida, o robô acessa os sites de e-commerce selecionados, realiza a busca dos produtos, coleta o valor do frete e armazena os resultados em um arquivo Excel.
